FAQs

1. What is the Document Object Model (DOM)?

The DOM is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content dynamically.

2. How do you select an element from the DOM?

You can use methods like `document.getElementById()`, `document.querySelector()`, and `document.getElementsByClassName()` to select elements from the DOM.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows a parent element to handle events for its children, even if the child elements are dynamically added later. This is useful for improving performance and managing dynamic content.

4. How do you manipulate an element's attributes and styles using the DOM?

You can use `element.setAttribute()`, `element.style`, and `element.classList` to manipulate attributes and styles of an element.
